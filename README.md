# BabyBloom Storefront Child Theme

Deploy this folder into `wp-content/themes/` on your WordPress install, activate the child theme, then assign menus and the homepage in WordPress admin.

## Git deployment

This theme is set up to work with GitHub plus Hostinger SSH deployment.

Use:

```powershell
.\deploy.ps1 "Describe the change"
```

First-time setup notes are in:

- `deploy.config.example.json`
- `HOSTINGER-SSH-SETUP.md`
